Script description

Script function of GlanceNet supports basic functions of javascript. (Compiler uses JerryScript.) In order to use GlanceNet's script function, you can cut or combine Packets using 'glance', a global object.
It will be easier to understand if you look at the example file (glance.js). (Glance.js script file can be downloaded from the Download menu.)

Script functions

Function : glance.getLengthSize = function()
Brief : size of packet length
Return : string or number
Description
length-size is the size of the packet length and means the length of the packet to be transmitted or received. Gets the received packet as much as the length of the packet. If there is no length part of the Packet, a value of '0' is returned. (In the case of HTTP protocol, there is no packet length. Therefore, the size of the packet length becomes '0'.)
Function: glance.getPacketLength = function(packet, issend, connid)
Brief : size of packet length
Return : string or number
Arguments
- packet (Uint8Array): Packet-Data
- issend (boolean) : send is true or false
- connid (string) : Connection-ID
Description
Returns the length of one packet. (Usually, the length of the packet is calculated. If there is no length, the total length of the packet to be displayed is returned.) Since HTTP protocol does not have a packet length and uses "\r\n" to separate packets, you need to use "\r\n" in the corresponding function to find the length of the entire packet.)
Function: glance.getPacketStruct = function(packet, issend)
Return : object
Arguments
- packet (Uint8Array): Packet-Data
- issend (boolean) : send is true or false
Brief : size of packet length
Description
The glance.getPacketStruct() function is called when the structure is displayed in the Packet Dialog. It must be returned in the form of Object, and the returned object is displayed on the Grid.
